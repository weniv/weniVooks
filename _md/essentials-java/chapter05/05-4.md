# 6. if-else 문의 성능 고려사항
- 조건의 복잡도: 간단한 조건을 먼저 검사하면 성능 향상에 도움이 될 수 있습니다.
- 조건의 빈도: 자주 발생하는 조건을 먼저 검사하면 평균 실행 시간을 줄일 수 있습니다.

```java
// 성능 최적화 예시
if (간단하고_자주_발생하는_조건) {
    // 처리
} else if (복잡하거나_덜_발생하는_조건) {
    // 처리
} else {
    // 기본 처리
}
```
