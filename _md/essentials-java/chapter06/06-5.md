# 6. 중첩 for 문
for 문은 다른 for 문 안에 중첩해서 사용할 수 있습니다.
```java
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        System.out.println("i = " + i + ", j = " + j);
    }
}
// 출력
// i = 1, j = 1
// i = 1, j = 2
// i = 1, j = 3
// i = 2, j = 1
// i = 2, j = 2
// i = 2, j = 3
// i = 3, j = 1
// i = 3, j = 2
// i = 3, j = 3
```
- 외부 for 문은 i를 1부터 3까지 증가시킵니다.
- 내부 for 문은 각 i 값에 대해 j를 1부터 3까지 증가시킵니다.
- 이 중첩 for 문은 총 9번 실행되며, (1,1), (1,2), (1,3), (2,1), ..., (3,3)과 같은 순서로 i와 j의 조합을 출력합니다.
