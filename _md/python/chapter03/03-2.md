# 1. 정수란?

0과 양의 정수, 음의 정수를 통틀어 정수라고 합니다. python에서 정수는 int라는 이름으로 불립니다. 한국어로 발음할 때에는 인트라고 부르고 다른 언어를 배우신 분들은 인티저라고 부르시는 분들도 있답니다.

## 1.1 정수 타입의 변수 생성

```python-exec
x = 10
y = -5
z = 0
print(type(x), type(y), type(z))
```

```python
# 출력
<class 'int'> <class 'int'> <class 'int'>
```

이 예제에서 `x`, `y`, `z`는 각각 10, -5, 0이라는 정수 값을 가지게 됩니다. python에서는 소수점이 있으면 정수로 구분하지 않으니 이를 주의해주세요. 10과 10.0은 다른 타입입니다.

## 1.2 정수의 사칙연산

정수는 기본적인 산술연산을 지원합니다. 이 챕터에서는 간단한 사칙연산만 해보도록 하겠습니다.

### 1.2.1 덧셈

```python-exec
x = 10
y = 20
result = x + y
print(result) # 출력 : 30
```

### 1.2.2 뺄셈

```python-exec
x = 10
y = 20
result = x - y
print(result) # 출력 : -10
```

### 1.2.3 곱셈

```python-exec
x = 10
y = 20
result = x * y
print(result) # 출력 : 200
```

### 1.2.4 나눗셈

```python-exec
x = 10
y = 20
result = x / y
print(result) # 출력 :
```

# 2. 진수

Python에서는 10진수 외에도 2진수, 8진수, 16진수로 정수를 표현할 수 있습니다. 아래 10진수와 2진수가 표현되는 방법을 기술하였습니다.

```python
# 10진수 731 이라는 숫자는 (7 * 10^2) + (3 * 10^1) + (1 * 10^0)이며 이는 10진수로 731입니다.
# 2진수  110 이라는 숫자는 (1 * 2^2) + (1 * 2^1) + (0 * 2^0)이며 이는 10진수로 6입니다.
x = 731
y = 0b110
```

진수는 지금 자세하게 공부하는 것보다는 프로젝트나 알고리즘에 필요할 경우 다시 찾아오셔서 보시는 것을 권해드립니다.

2진수는 알고리즘 문제로 자주 출제가 됩니다. 16진수는 컴퓨터에서 색 표현으로 많이 사용합니다. `000000`은 흰색이고, `ffffff`는 검은색입니다.

## 2.1 2진수 (Binary)

2진수는 0과 1로만 구성되며, Python에서는 숫자 앞에 `0b`를 붙여서 2진수로 표현합니다.

```python-exec
x = 0b110
print(x)
print(type(x))
```

이 예제에서 `x`는 2진수 110을 10진수로 변환하여 6이 출력됩니다. `type(x)`는 `<class 'int'>`로, `x`의 데이터 타입이 정수임을 확인할 수 있습니다.

```python
# 출력
6
<class 'int'>
```

## 2.2 8진수 (Octal)

8진수는 0부터 7까지의 숫자로 구성되며, Python에서는 숫자 앞에 `0o`를 붙여서 8진수로 표현합니다.

```python-exec
x = 0o15
print(x)
print(type(x))
```

이 예제에서 `x`는 8진수 15를 10진수로 변환하여 13이 출력됩니다. `type(x)`는 `<class 'int'>`로, `x`의 데이터 타입이 정수임을 확인할 수 있습니다.

```python
# 출력
13
<class 'int'>
```

## 2.3 16진수 (Hexadecimal)

16진수는 0부터 9까지의 숫자와 A(10)부터 F(15)까지의 알파벳으로 구성되며, Python에서는 숫자 앞에 `0x`를 붙여서 16진수로 표현합니다.

```python-exec
x = 0x1a
print(x)
print(type(x))
```

이 예제에서 `x`는 16진수 1a를 10진수로 변환하여 26이 출력됩니다. `type(x)`는 `<class 'int'>`로, `x`의 데이터 타입이 정수임을 확인할 수 있습니다.

```python
# 출력
26
<class 'int'>
```

## 2.4 진수 변환 함수

Python에서는 `bin()`, `oct()`, `hex()` 함수를 사용하여 10진수를 각각 2진수, 8진수, 16진수로 변환할 수 있습니다.

```python-exec
x = 26
print(bin(x))
print(oct(x))
print(hex(x))
```

```python
# 출력
0b11010
0o32
0x1a
```

이 예제에서 `x`는 26이고, `bin(x)`는 '0b11010', `oct(x)`는 '0o32', `hex(x)`는 '0x1a'를 출력합니다. 이렇게 변환된 값은 int built-in-function을 사용하여 10진수로 다시 변환할 수 있습니다.

```python-exec
b = int('0b11010', 2)
o = int('0o32', 8)
h = int('0x1a', 16)
print(b, o, h)
```

출력값은 모두 26입니다.

```python
# 출력
26
26
26
```

# 3. 메서드

메서드는 자료형(정수, 실수, 문자열, 리스트 등)이 할 수 있는 일(함수)입니다. 예를 들어, 문자열 객체는 문자열을 대문자로 바꾸는 일을 할 수 있습니다. 이러한 일을 하는 함수를 '메서드'라고 부릅니다.

예를 들어, 'hello'라는 문자열이 있다면, 'HELLO'로 바꾸는 일을 하는 함수가 있습니다. 이 함수를 사용하려면 'hello'.upper()라고 씁니다. 여기서 'upper'는 'hello'라는 문자열 객체가 할 수 있는 일 중 하나인 메서드입니다.

다만 숫자에서 사용할 수 있는 메서드는 몇 개 되지 않으며, 그마저도 사용할 일이 거의 없습니다. 이와 같은 이유로 뒤에서 다룰 자료형인 float와 complex에서는 메서드를 다루지 않겠습니다.

여기서는 메서드를 어떻게 확인하고, 어떻게 활용하는지 살펴보도록 하겠습니다.

```python-exec
x = 5
print(dir(5))
```

위처럼 `dir`이라는 함수를 통해 자료형이 가진 메서드를 확인할 수 있습니다. 여기서 언더스코어(`_`) 2개가 붙어있는 것들을 매직 메서드라고 하고, 언더스코어가 없는 것들을 메서드라고 합니다.

```python
# 출력
['__abs__',
 '__add__',
 ... 생략 ...
 '__xor__',
 'as_integer_ratio',
 'bit_count',
 'bit_length',
 'conjugate',
 'denominator',
 'from_bytes',
 'imag',
 'numerator',
 'real',
 'to_bytes']
```

매직 메서드는 클래스 파트에서 다루게 됩니다. 가볍게 언급을 해드리자면 매직 메서드는 속성입니다. 예를 들어 아래와 같은 더하기 연산이 되는 것은 `__add__`가 있기 때문입니다.

```python-exec
x = 5
print(x + x) # 출력 : 10
```

앞으로 자료형에서는 이 매직 메서드를 제외하고 일반 메서드, 그 중에서도 자주 사용되는 일반 메서드를 살펴보도록 하겠습니다.

정수 자료형에 메서드 중에서는 일반적으로 사용되는 메서드는 없는데요. 그래도 메서드가 어떤 일을 하는지 알기 위해 `bit_count`메서드를 살펴보도록 하겠습니다.

```python-exec
x = 5
x.bit_count() # 출력 : 2 101 -> 1이 두개 2
```

```python-exec
x = 4
x.bit_count() # 출력 : 1 100 -> 1이 한개 1
```

5는 2진수로 표현했을 때 `101`이고, 4는 2진수로 표현하면 `100`입니다. 여기서 1의 갯수를 카운팅한 것입니다.

여기서 주의깊게 봐야 할 점은 x뒤에 있는 점(Dot)입니다. **점 표기법(Dot Notation)** 이라고 하는데요. 이렇게 점을 찍으면 해당 자료형이 가진 메서드에 접근할 수 있습니다.
