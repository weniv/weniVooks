# 1. 문자열 연결하기

문자열을 연결할 때는 `+`이 아닌 `||`를 사용합니다. 아래 코드를 실행해보세요.

```sql
-- SELECT 상품명 + 설명 from 상품;
SELECT '안녕하세요' || ' ' || 'SQL' || ' ' || '캠프';
```

# 2. 문자열 연결 활용

아래와 같이 상품명과 설명을 연결하여 출력해보세요.

```sql
SELECT 상품명 || ': ' || 설명 from 상품;
```

SELECT 상품명 AS 제품이름, 상품명 || '가 ' || (가격*0.2) || '할인되어 ' || (가격*0.8) || '에 판매합니다!' AS 홍보문구 FROM 상품;
