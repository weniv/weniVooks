# 수업목표

- 분석에 필요한 열자료를 Select문을 활용해 가져올 수 있습니다.
- 열자료를 함수를 활용해 간단하게 집계할 수 있습니다.
- 열자료에서 중복되는 데이터를 제거할 수 있습니다. (Distinct 활용)

# 개요

이제 원하는 테이블을 가져왔다면 내가 원하는 자료를 가져오는 방법을 알아야할텐데요. `From` 문에서 잠깐 살펴봤던 `Select` 문을 활용해서 필요한 자료를 가져오고, 함수를 활용해 집계를 하거나(덧셈, 평균 등), 중복을 제거하는 등 필수적인 SQL 문법에 대해서 배우게 됩니다.

# 내가 원하는 데이터를 가져오는 - SELECT

**SQL 2원칙에서 배울 `SELECT`** 문은 SQL에서 가장 기본적인 명령어 중 하나로, 데이터베이스에서 특정 데이터를 조회할 때 사용됩니다. 간단히 말해서, **`SELECT`**는 테이블에서 원하는 열(컬럼)을 선택하여 출력하는 역할을 합니다.

## 2.2 머릿속으로 미리 결과를 예상해보기

앞서 우리는 주문테이블에서, 주문ID, 배송주소, 총주문금액을 잘 추출했는데요. 

| 주문ID | 배송주소 | 총주문금액 |
| --- | --- | --- |
| 1001 | 서울특별시 강남구 한류로 123 | 1697000 |
| 1002 | 경상남도 거제시 바닷소리로 1616 | 69900 |
| 1003 | 광주광역시 서구 무궁화로 202 | 244300 |
| 1004 | 강원도 강릉시 동해대로 707 | 65000 |

만약 위와 같이 4개의 데이터가 있다고 했을때 데이터의 개수, 합, 평균은 어떻게 될까요? 

데이터의 개수를 파악할 때는 주문ID의 열에서 파악하는 게 훨씬 편리하겠죠.

주문ID 기준으로 개수는 1001부터 1004까지 4개, 

총주문금액의 합은 (1,697,000 + 69,900 + 244,300 + 65,000)으로 2,076,200

총주문금액의 평균은 2,076,200에 4를 나눈 519,050이 나올것입니다. 

데이터의 수가 적으면, select로 필요한 열을 불러서 일일이 세고, 더하고, 평균값을 내면 될 텐데요. 데이터가 커지면 매번 이렇게 일일이 계산할수 없기 때문에 SQL의 함수를 활용해야합니다. 

개수를 세는 방법은 count(), 합을 구할 때는 sum(), 평균을 구할 때는 average()를 입력합니다. 

그러면 한번 실습해보겠습니다.

# Summary

우리는 이번시간에 Select에 대해서 배웠습니다.

1. 우리가 원하는 열자료를 선택하고 

`Select 열자료이름`

1. 2개 이상의 열자료를 선택하는 방법 

`Select 열자료이름1, 열자료이름2, 열자료이름3`

1. 열자료의 갯수, 합, 평균을 구하는 방법

`Select count(열자료이름(문자/숫자)), 
        sum(열자료이름(숫자형만), 
        avg(열자료이름(숫자형만)`

1. 열자료의 중복을 제거하는 방법

`Select distinct(열자료이름)`

이로써 우리는 분석을 위해 필요한 열자료를 선택하고, 집계를 하고, 중복을 제거하는 방법에 대해 알아봤습니다.

## Tip. 기본 쿼리문 실행

select 문은 From이 없이도 SQL을 활용해 연산을 수행할 수 있습니다. 계산기 역할과 비슷하다고 보시면 됩니다.

- 기본적으로 쿼리문은 소문자로도 대문자로도 작성할 수 있습니다.
- 숫자를 그대로 출력할 수 있습니다.
    
    ```sql
    select 1;
    ```
    
- 숫자를 연산하여 출력할 수 있습니다. 연산자 우선순위가 지켜집니다.(곱하기, 나누기, 나머지가 먼저 연산됩니다.)
    
    
    | + | 덧셈 |
    | --- | --- |
    | - | 뺄셈 |
    | * | 곱셈 |
    | / | 나눗셈 |
    
    ```sql
    select 1+1;
    ```
    
    ```sql
    select 10-20;
    ```
    
    ```sql
    select 3*6;
    ```
    
    ```sql
    select 7/3;
    ```
    
- 문자열 그대로를 출력할 수 있습니다. 문자열을 더해보세요.
    
    ```sql
    select 'hello world';
    ```
    
    ```sql
    --문자열은 더하기가 안됩니다.
    select 'hello'+'world';
    ```
    
- 여러개로 출력하고 싶다면 아래처럼 사용해야 합니다.
    ```sql
    select 'hello', 'world';
    ```
