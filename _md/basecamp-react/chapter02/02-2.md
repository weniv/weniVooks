# 1.1. React 실행

1. 터미널을 열고 아래 명령어를 실행하면 React로 이루어진 프로젝트를 생성할 수 있습니다.

:::div{.callout}
아래 명령어를 사용하기 위해서는 Node 14.0.0 이상 또는 npm 5.6 이상의 버전이 필요 합니다. npx는 npm의 도구이며 npm의 5.2.0 이상에서 지원합니다.

:::

```bash
npx create-react-app my-app

# my-app이라는 폴더에 React와 모듈들이 함께 설치됩니다.
```

create-react-app에 대해 더 알고 싶다면 아래 링크를 참고해 주세요.

::a[create-react-app이 무슨 일을 할까?]{class='btn-link' href="https://velog.io/@devsaza/cra" target="_blank"}

2. 프로젝트를 생성했으니 아래 명령어를 입력하여 이제 React를 실행해 볼까요?

```bash
cd my-app
npm start
```

3. 터미널에 다음과 같은 문구와 웹페이지가 나타났다면?  React 실행에 성공하셨습니다!

::img{width="300" alt="React Execution Completion" src="/images/basecamp-react/chapter02-2/chapter02-2-4.png"}

::img{width="300" alt="React Execution Screen" src="/images/basecamp-react/chapter02-2/chapter02-2-5.png"}

:::div{.callout}
**혹시 에러가 났나요?**

1. 아래 명령어를 실행하세요.

```bash
npx create-react-app@latest my-app
```

2. 아래와 같은 에러가 발생했다면, `npm uninstall -g create-react-app` 명령어를 입력하세요.

::img{width="300" alt="Error Message" src="/images/basecamp-react/chapter02-2/chapter02-2-6.png"}

3. 설치가 계속 되지 않을 경우는 아래 명령어를 통해 캐쉬를 비워 주세요.

```bash
npm cache verify
npm cache clear --force
```

:::

# 1.2. Project 작업

:::div{.callout}
자동 생성된 파일들 중, 아직은 필요하지 않은 파일들은 지워주도록 하겠습니다. 아래 사진의 파일들만 남겨 주세요. (파일 삭제 후에는 의존하고 있는 Code들의 수정이 이루어져야 하는데, 이 과정은 뒤에서 진행하도록 하겠습니다.)

:::

::img{width="300" alt="File Lists" src="/images/basecamp-react/chapter02-2/chapter02-2-7.png"}

```jsx
<!DOCTYPE html>
<html lang="ko">

<head>
  <meta charset="utf-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1" />
  <title>React App</title>
</head>

<body>
  <p>test1</p>
  <div id="root"></div>
</body>

</html>
```
`index.html`

```jsx
function App() {
  return (
    // JSX 문법
    <div className="App">
      <h1>test2</h1>
    </div>
  );
}

export default App;
```
`App.js` `(메인 페이지)`

```jsx
import React from 'react';
// import ReactDOM from 'react-dom'; //구버전
import { createRoot } from 'react-dom/client';
// App.js의 App을 가져오는 것을 확인할 수 있습니다.
import App from './App';

const container = document.getElementById('root');
const root = createRoot(container);
// root의 render 부분이 요소로 바뀌어서 html 안으로 들어갑니다.
root.render(<App tab="home" />);
```
`index.js` (메인 페이지를  index.html에 보여지게 함)

:::div{.callout}
**document가 index.html와 연결되는 방법**
웹팩이 작성한 코드를 하나로 묶어주는 역할을 해줌으로써 연결이 됩니다. 기본적으로 설치가 되어 있고, 브라우저에서 소스코드를 보면 추가하지 않았던 bundle.js를 보면 알 수 있습니다. 웹팩에 대해 더 자세히 알고 싶으시다면 아래 링크를 참고해 주세요.

::a[Create-React-App의 Webpack 기본 설정 살펴보기]{class='btn-link' href="https://maxkim-j.github.io/posts/cra-webpack-config/" target="_blank"}

`※주의※` `npm run eject` 하면 되돌리기가 불가능한 점 유의해 주세요!

:::

## 1.3.2. 코드 설정

1. GitHub Page를 만들어 주세요.
VSC를 열어 `package.json`에 아래 코드를 추가합니다.

```jsx
{
	"homepage":"여러분 GitHub page url",
	...
}
```

2. 터미널에서 단축키 command + c 를 눌러 실행을 취소해 주세요.
3. `npm run build` 명령어를 실행하면 build 폴더로 resource 파일들이 업로드 됩니다.
4. build 안에 있는 파일 모두를 GitHub pages에 업로드하면 됩니다.
