# 탐색적 자료 분석

- 데이터를 분석하기 전에 그래프나 통계적인 방법으로 데이터를 직관적으로 바라보는 과정
- 데이터를 있는 그대로 바라보는데 중점을 맞추어 데이터가 가지고 있는 의미를 다양한 각도로 바라보고 이해

# EDA 목적

- 데이터 수집 의사를 결정
- 데이터 유형에 맞는 모델을 선택
- 변수들 사이의 관계를 파악
